name: check-keyword
description: Check the SEO potential of a keyword for web site building and finding its relevant variants.
---

## Tools to use

- Semrush: see search volume, keyword difficulty, CPC, related keywords etc.
- Google trends: see trends and related queries
- Search on X: sentiment analysis and community insights
- Search on Reddit: sentiment analysis and community insights
- Inspect SERP: understand search intent and competition
- Ahrefs: check keyword difficulty for cross-checking with Semrush

NEVER rely on a single tool, always cross-check with multiple sources to get a more accurate and comprehensive understanding of the keyword's potential.

## Output

- Add the result to `./notes/keyword-research.csv`, fill in the information where applicable. NEVER delete or modify existing entries for other keywords, only append new rows or update entries for the current keyword.
- A short report
