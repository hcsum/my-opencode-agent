# 韭俱乐部 黑话 / 缩写词典

Read during decode. When you confirm a new term (from a report or via web-access), append it with a date. Keep entries one line. Mark anything still unconfirmed with `⚠`.

## 操作 / 仓位语言

- 右侧 — 趋势已确认(站上关键均线/突破/有量配合),顺势参与。
- 左侧 — 趋势未立,基于估值或位置在确认前提前布局。
- 右侧抱团 — 资金形成合力、跌下去有强承接意愿的板块;抱团松动=承接意愿下降、合力变弱,散户宜规避观望。
- 鬼故事 — 利空/吓人叙事(常带问号,未必兑现)。
- 杀杠杆 — 去杠杆、强制平仓踩踏。
- 反脆弱 — Nassim Taleb 概念,仓位结构越波动越受益。
- 上影线 — K线见顶信号;「牛市最后一根上影线」=见顶警惕。
- 老登(股) — 权重蓝筹/旧经济大票(「老头股」)。
- 韭 / 韭补 — 韭菜=散户;「一跌韭补」=一跌就盲目补仓摊低(被劝阻)。
- 汪汪队 — 国家队/维稳资金(Paw Patrol 谐音)。
- 存款搬家 — 居民储蓄迁入股市。
- 险资 — 保险资金。
- 杠铃(策略) — 两端配置(防守端 + 高弹性端),中间不配。
- 砍人举债维持财报 — 裁员 + 举债来撑住业绩(非「举报」)。
- 四档框架 — 防守 / 进攻配置 / 进攻波段 / 进攻投机;账户头部「X进攻/Y防守/Z现金」中,进攻 = 配置+波段+投机之和。

## 主线缩写

- 中光韩存美费半 — 中国光模块 / 韩国存储 / 美国费城半导体(SOX 费城半导体指数);算力链必持主线。
- 光通信 — 光模块 / PCB(印刷电路板)/ CCL(覆铜板)/ 光纤,北美算力叙事。
- 存储映射 — 港韩/全球芯片/中韩半导体的存储涨价映射标的。
- 中概 — 中国互联网中概股(腾讯/阿里/美团/拼多多等),A股账户里经港股通或 QDII 中概互联ETF 持有。

## 宏观 / 机构缩写

- PCE — Personal Consumption Expenditures 个人消费支出物价指数(美联储最看重的通胀锚,核心 PCE 对应 2% 目标)。
- CPI / PPI — 消费者 / 生产者物价指数。
- PMI — Purchasing Managers Index 采购经理人指数。ISM=美国供应管理协会(制造业/服务业)。ADP=ADP 就业报告(小非农)。
- GDP — Gross Domestic Product 国内生产总值。
- MLF — Medium-term Lending Facility 中期借贷便利;「到期续作」力度与利率是松紧信号(超额续作=净投放偏松,缩量=净回笼偏紧)。
- RatingDog PMI — = 原财新 PMI,2025/8 改名,S&P Global 仍做调查。
- FOMC — Federal Open Market Committee 联邦公开市场委员会。
- IV — Implied Volatility 隐含波动率。
- ZF — 政府(拼音首字母,如「美国ZF再关门」=政府停摆)。

## 科技 / 产业

- M7 — Magnificent 7 美股七巨头。
- CTA — Commodity Trading Advisor 商品交易顾问(趋势跟踪量化基金,程序化买卖放大波动)。
- CSP — Cloud Service Provider 云服务商。
- NeoCloud — 新型 GPU 算力云(CoreWeave 类);「融资利率下降」指其信用利差收窄,非美联储政策利率。
- hyperscaler — 超大规模云厂商。
- ARR — Annual Recurring Revenue 年度经常性收入。
- fabless — 无晶圆厂芯片设计。
- NAND — 闪存;长江存储(YMTC)上市/扩产 = 供过于求鬼故事风险。
- MLCC — Multilayer Ceramic Capacitor 多层陶瓷电容。
- PMIC — Power Management IC 电源/功率芯片(报告写「电芯片」指此,非电池电芯)。
- SiC — Silicon Carbide 碳化硅。
- 情绪板块 — 靠情绪/题材/资金驱动、无基本面支撑的投机板块(meme/概念);流动性收紧环境难持续炒,只宜短线触底突破时博反弹。
