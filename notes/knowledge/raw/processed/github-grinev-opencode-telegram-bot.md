# Opencode Telegram Bot

**Author:** grinev  
**Repository:** https://github.com/grinev/opencode-telegram-bot  
**Source:** [awesome-opencode](https://github.com/awesome-opencode/awesome-opencode) curated list

## Description

Telegram bot client for OpenCode CLI. Run and monitor AI coding tasks from your phone while everything runs locally on your machine.

## Key Characteristics

- **Platform:** Telegram Bot API
- **Integration:** OpenCode CLI
- **Architecture:** Bridge / client wrapper around OpenCode CLI
- **Deployment:** Local machine (everything runs locally)
- **Use case:** Remote monitoring and control of OpenCode sessions from mobile devices

## Relation to User's Project

Similar to the user's `opencode-agent` project (this repo), which is also a Telegram bridge for OpenCode. The user's project is described as a "Telegram bridge for a shared OpenCode server" with skills, notes, and Gmail integration.

## Differences from User's Project

- grinev's version is a direct Telegram bot wrapper for personal/local OpenCode CLI usage
- User's project appears to be a shared/multi-user bridge with additional features (skills, notes, Gmail)
